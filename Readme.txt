This application is using Node Version 8.9.4 and Angular 5

Steps:

1. Run Web api project
2. Run Mean Stack project in GIT -> Mean folder
3. GOTO path till NodeJS in cmd and execute command command -> 
	> node index.js

To Register and Login into this application.
1. User can register using the web api project but in that case Login is yet to be implemented
2. User can also register using MEAN stack project and can look corresponding user details by installing MongoCompase application
To do this
	> Change Angular rootURL to to point to either Node JS or IIS web server  
	> One this is done execute npm run watch for angular project
